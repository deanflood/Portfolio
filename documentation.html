Overview

  • Website was developed from scratch, using no template for the bootstrap or CSS
  • Website is hosted on Github Pages at the URL www.deanflood.ie
  • Github Repo for the project is on my Github
  • Bootstrap was used to ensure the website was responsive, and using Bootstraps Grid layout, the website looks attractive across various screen sizes.

Interactive Elements

  • To make the website react to users, I used a variety of JQuery and CSS elements
  • Modals are used to enlarge my user profile image, as well as to display a Contact Screen when the user selects the button as shown
  • Interactive elements such as the Accordion and navbar were based on JQUERY elements found at jquery.com
  • Jquery Accordion was used for displaying experience and results, and Jquery Tabs were used for the main content on screen.
  • W3 Slideshow CSS was as a base to display my project work

Dynamic Elements

  • I introduced two separate areas where the data is dynamic instead of being hardcoded
  • This was done using Angular JS, extracting the data from JSON Objects and inserting it into the page.
  • Both the skill set and result sections are dynamically written to the page.
  • As these are displayed as tables with more than one colume, my logic for the loop had to ensure the data was inserted correctly.
  • I achieved this by using modulus calculation
